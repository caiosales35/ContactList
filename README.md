# Contact List

Projeto de agenda de contatos; criado com o objetivo de iniciar no Ruby on Rails.

## Requisitos Necessários:

* Ruby 2.3.0 ou Superior
* PostgreSQL
* Rails 5

## Funcionalidades

- Cadastro de usuários
- Login de Usuário
- Gerenciamento de contatos pessoais
